# ⚽ Football Tournament Manager

A simple football tournament manager built using Python.

This project allows you to manage football tournaments by adding teams, generating fixtures, and creating knockout brackets.

---

## 🚀 Features

- Add teams
- Generate fixtures automatically
- Create group stages
- Generate knockout brackets
- Points table system
